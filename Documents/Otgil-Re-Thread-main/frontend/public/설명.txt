public 폴더
역할: 이 폴더 안의 파일들은 빌드 과정에서 특별히 처리되지 않고, 그대로 최종 결과물 폴더(보통 build 또는 dist)로 복사됩니다. 웹 서버가 직접 접근해야 하는 정적 파일들을 여기에 둡니다.
주요 파일:
index.html: 가장 중요한 파일입니다. 빈 <div> 태그(현재 코드에서는 <div id="root"></div>)를 가지고 있는 HTML 템플릿입니다. 웹팩(Webpack)이나 Vite 같은 번들러가 src 폴더의 모든 React 코드를 하나의 JavaScript 파일로 합친 후, 이 index.html 파일에 자동으로 삽입해 줍니다. 최종적으로 브라우저가 열게 되는 파일입니다.
favicon.ico: 브라우저 탭에 표시되는 작은 아이콘입니다.
robots.txt: 검색 엔진 크롤러에게 어떤 페이지를 수집하고 어떤 페이지를 수집하지 말아야 할지 알려주는 파일입니다.
기타 이미지, 폰트 등: 소스 코드에서 import하여 사용하지 않고, URL로 직접 접근해야 하는 파일들을 이곳에 둘 수 있습니다.
